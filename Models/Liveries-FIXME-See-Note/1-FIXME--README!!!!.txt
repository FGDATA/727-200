These liveries need the engines mapped. See the engine painkit in the Painkit Folder.

Also, livery XMl needs the following:

<?xml version="1.0"?>

<PropertyList>
	<sim>
		<model>
			<livery>
				<name type="string">Boeing 727</name>
				<texture>Liveries-ADV/BOE.png</texture>
			</livery>
			<eng>
				<texture>Liveries-ADV/BOE-eng.png</texture>
			</eng>
		</model>
	</sim>
</PropertyList>

Replace the BOE with the texture name.
